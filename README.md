# Quick start

Install:

```
git clone git@github.com:gabriel-cardoso-oliveira/ranking-kart.git
```
```
cd ranking-kart/
```
```
npm install
```

Test:

In the project directory, you can run:
```
npx react-native start
```
In another terminal:
```
npx react-native rum-android
```
