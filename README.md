# Quick start

Install:

```
git clone git@github.com:gabriel-cardoso-oliveira/ranking-kart.git
```
```
cd ranking-kart/
```
```
npm install
```

Test:

```
npm react-native start
```
In another terminal:
```
npm react-native rum-android
```
